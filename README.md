# AYIRAW TECH LLP - Web Platform

Official website for AYIRAW TECH LLP, a leader in B2BD2C food tech manufacturing. Specialized in high-volume culinary automation, specifically the **Automatic Puttu Maker**.

## 🚀 Quick Start (Deployment on Render)

This project is optimized for **Render Static Site Hosting**.

1. **Create a GitHub Repo**: Name it `ayiraw-tech-web`.
2. **Push these files**:
   - `index.html` (Home)
   - `about.html` (About)
   - `products.html` (Puttu Maker)
   - `leadership.html` (Mahesh P.G. Profile)
   - `css/tailwind.css`
   - `scripts/main.js`
3. **Connect to Render**:
   - Select "Static Site".
   - Connect your GitHub repo.
   - **Build Command**: `None` (or `npx tailwindcss -i ./css/tailwind.css -o ./public/output.css` if using CLI).
   - **Publish Directory**: `.` (Root).

## 📁 Repository Structure

- `index.html`: Landing page highlighting "Industrial Precision in Food Tech".
- `products.html`: Deep dive into the 50-unit Puttu Maker specs.
- `leadership.html`: Profile of Director Mahesh Perathu Gireeshkumar & patent history.
- `public/`: Images and static assets.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **Tailwind CSS**: Industrial design system utility-first styling.
- **Vanilla JS**: Performant interactions and mobile navigation.
